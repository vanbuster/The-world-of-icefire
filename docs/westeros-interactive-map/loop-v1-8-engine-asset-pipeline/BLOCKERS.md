- 本机是否安装 Unity / Unreal / Blender 尚未确认；若未安装，V1.8 应用程序化 / AI / Web 资产模拟引擎资产管线。
- 真实 Unity WebGL 原型可能需要额外工程目录、构建时间和浏览器包体验证，不应阻塞 C 方案主线。
- 公开发布前，任何引擎素材、第三方模型、AI 生成贴图仍需来源与授权记录。
